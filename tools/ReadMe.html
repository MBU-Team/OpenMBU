Torque Shader Engine

Tools Release ReadMe

Welcome to the GarageGames Torque Shader Engine tools. This is a release of the torque tools binaries. This release does not include the SDK source code, nor the test application. SDK source code is only available online through CVS. If you are a registered SDK developer and wish to download the source, please see the Using CVS section in the Torque SDK documentation for more information. The test application release is also available from the private SDK download page.

Tools

These tools are currently only released as Win32 binaries. None of the tools are available on the Mac or Linux/OpenBSD platforms at this point, except of course the in-game tools, which include the mission and GUI editors.

To get started with the tools, visit the online Torque Game Engine Documentation page. This page contains links to all available test application and tool documentation. If your a registered developer, the SDK home page has a "Getting Started" section for developers. This file is only a simple overview of the contents of this release, all documentation is available online and linked from the two pages listed ubove.

Tools located in the \Torque Tools directory:

ms2dtsExporter.dllDTS exporter for MilkShape
map2DTSExporter.dleMAP Exporter for 3D Studio Max 4.0
buildWad.exeWAD file builder for use with WorldCraft
map2dif.exeWorldCraft & Quake .MAP converter
texture2bm8.exePalettizes .JPEG files for use with low end video hardware

There are two versions of the 3D Studio Max exporter tools (.dle files) one set for Max 3.1 and the other for Max 4.0, these are located in the \max3 and \max4 sub-directories:

max2DTSExporter.dleDTS Exporter for 3D Studio Max 3.0
max2DTSExporter.dleDTS Exporter for 3D Studio Max 4.0
